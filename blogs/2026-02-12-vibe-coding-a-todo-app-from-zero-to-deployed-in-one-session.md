<p>How fast can you go from an empty folder to a fully deployed full-stack app? In this post, I'll walk through a live vibe coding session where I pair-programmed with an AI agent (Claude Code) to build a todo app using <strong>React</strong> for the frontend and <strong>Codehooks.io</strong> for the backend — all deployed to the cloud in minutes.</p>
<p>Every prompt I typed, every decision the agent made, and every line of code it wrote is documented here. Follow along and try it yourself.</p>

<h3 class="anchor anchorWithStickyNavbar_LWe7" id="agent-reasoning">Agent reasoning<a class="hash-link" href="https://codehooks.io/blog/vibe-coding-todo-app#agent-reasoning" title="Direct link to Agent reasoning">​</a></h3>
<blockquote>
<p>"The key decision here is using <code>crudlify()</code> instead of writing manual routes. One line — <code>app.crudlify({ todos: TodoSchema }, { prefix: '/api' })</code> — auto-generates a complete REST API with Zod validation. That single line gives us:</p>
<ul>
<li><code>GET /api/todos</code> — list all todos</li>
<li><code>POST /api/todos</code> — create a todo (validated against the Zod schema)</li>
<li><code>GET /api/todos/:id</code> — get one todo</li>
<li><code>PUT /api/todos/:id</code> — replace a todo</li>
<li><code>PATCH /api/todos/:id</code> — partial update (perfect for toggling <code>completed</code>)</li>
<li><code>DELETE /api/todos/:id</code> — remove a todo</li>
</ul>
<p>The <code>app.static()</code> block at the end serves the React SPA. The <code>notFound: '/index.html'</code> fallback enables client-side routing."</p>
</blockquote>
